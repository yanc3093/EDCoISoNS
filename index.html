Event Driven Classification of Images
- Social or Non-Social

About this project

This project was my Final Year Project (FYP). It was a group work completed in October 2022.

Different from the ordinary school assignments, this project was supervised by a lecturer from the School of Information Technology at Monash University Malaysia, who acted as the project client or customer.

The objective of this project was to design and create a system that classifies the images based on the depicted events into 2 broad categories: social and non-social. Examples of social events are graduation, birthday, and wedding.

Based on the client's requirement, the system should comprise 3 key deliverables:

  • An image database with at least 3000 images in each of the 2 categories
  • An event driven image classifier (backend)
  • A graphical user interface (frontend)

The product acceptance criterion was that the event driven image classifier must achieve a prediction accuracy of at least 0.75 or 75%.

Project outcomes

This project was executed in 4 major phases, which followed the waterfall project management methodology:

  • Requirements phase, during which we collected and documented the client's requirement.
  • System design phase, during which we carried out research on the project related topics and derived a feasible solution.
  • Implementation phase, during which we implemented the solution according to the planned schedule.
  • Verification phase, during which we checked whether our product meets the client's requirement and acceptance criterion stipulated at the beginning.

Particularly, the implementation phase comprised 3 subphases:

  • Construct an image database.
  • Build an event driven image classifier.
  • Create a graphical user interface.

To construct an image database, my teammates and I tried to look for a suitable one on the internet. Eventually, we selectively combined the images from the following 4 datasets:

  • Social Event Image Dataset (SocEID)
  • Animals10 Dataset
  • Indoor Scene Recognition Dataset
  • Intel Image Classification Dataset

As for the event driven image classifier, it is a Support Vector Machine (SVM) that classifies the images using the Histogram of Oriented Gradients (HOG) feature, which represents the shapes of the objects within an image.
